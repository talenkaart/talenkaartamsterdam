# Talenkaart Applicatie
Talenkaart applicatie Amsterdam

## Data beheren

De data voor deze applicatie staat in een Excel-bestand op GitHub. De bestanden zijn te vinden in de map [`static/TALENKAART_DATA`](./static/TALENKAART_DATA).

### Data bekijken en downloaden

Klik op een bestand en kies vervolgens de downloadknop (het icoontje met een pijltje naar beneden) om het te downloaden. Dit kan iedereen zonder account.

### Data aanpassen

Om data aan te passen heb je een **GitHub-account nodig met toegang tot deze repository**. Heb je dat nog niet? Neem dan contact op met: contact@degeomaten.nl

Heb je toegang? Volg dan deze stappen:

1. Ga naar de map [`static/TALENKAART_DATA`](./static/TALENKAART_DATA)
2. Klik op het Excel-bestand dat je wilt aanpassen
3. Download het bestand via de downloadknop rechtsboven
4. Pas de data aan in Excel op je eigen computer
5. Ga terug naar dezelfde pagina op GitHub
6. Klik op het uploadicoontje om je aangepaste bestand te uploaden
7. Scroll naar beneden en klik op **"Commit changes"** om de wijzigingen op te slaan

De applicatie laadt automatisch de nieuwe data zodra de wijzigingen zijn opgeslagen.
